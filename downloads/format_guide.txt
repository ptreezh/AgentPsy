# 技术白皮书格式和转换说明

## 提供的格式

我们提供了以下格式的技术白皮书：

### 文本格式 (.txt)
- 纯文本格式，适用于所有平台
- 可直接阅读，兼容性最好
- 文件大小最小

### Markdown格式 (.md)
- 结构化标记格式
- 易于阅读和转换
- 可在GitHub、GitLab等平台直接渲染
- 保持文档结构

## 转换为PDF或Word格式

### 方法一：使用Pandoc (推荐)
Pandoc是一个强大的文档转换工具：

```bash
# 转换为PDF
pandoc -f markdown -t pdf -o output.pdf input.md

# 转换为Word
pandoc -f markdown -t docx -o output.docx input.md
```

### 方法二：使用Microsoft Word
1. 打开Word
2. 选择"文件" → "打开"
3. 选择.md文件或直接粘贴内容
4. 使用"文件" → "另存为"保存为.docx格式

### 方法三：使用Google Docs
1. 访问 docs.google.com
2. 粘贴文档内容
3. 选择"文件" → "下载" → 选择所需格式

### 方法四：在线转换工具
- https://www.markdowntopdf.com/
- https://www.convertio.co/md-pdf/
- https://wordtohtml.net/markdown-to-word/

## 格式选择建议

- **阅读**: 使用.txt格式，兼容性最好
- **编辑**: 使用.md格式，保持文档结构
- **打印**: 转换为PDF格式，格式最为稳定
- **协作**: 转换为Word格式，便于评论和修改

## 文件命名规则

- 中文版技术白皮书: 
  - tech_whitepaper_cn.txt
  - tech_whitepaper_cn.md
- 英文版技术白皮书:
  - tech_whitepaper_en.txt
  - tech_whitepaper_en.md

所有文档内容完整，您可以根据需要选择合适的格式使用。
