Quick Start Guide

Get up and running with AgentPsy's assessment tools in minutes.

Introduction to AI Personality Assessment

AgentPsy provides scientifically-based tools for assessing personality traits in AI models. Our methodology requires extensive testing with multiple parameter combinations to establish reliable personality profiles.

Important: AI personality assessment requires thousands of test iterations to establish statistically significant results. A single evaluation is not sufficient for reliable personality profiling.

Basic Setup

1

Install Dependencies

Install the AgentPsy framework and required dependencies:

pip install agentpsy-client
pip install transformers torch
pip install numpy pandas scikit-learn
pip install python-dotenv
2

Configure Environment

Set up your API credentials in a .env file:

# .env file
PROVIDER=openai
OPENAI_API_KEY=your_openai_key
# Or for other providers:
# GEMINI_API_KEY=your_gemini_key
# ANTHROPIC_API_KEY=your_anthropic_key
3

Initialize the Framework

Create a basic assessment framework instance:

from agentpsy import AssessmentFramework
import os

# Load environment variables
from dotenv import load_dotenv
load_dotenv()

# Initialize the assessment framework
framework = AssessmentFramework(
    model="gpt-4",
    evaluation_methods=["big_five", "cognitive_stability", "cognitive_trap"]
)
4

Run a Basic Assessment

Perform a preliminary assessment to test setup:

# Sample text to assess
sample_text = """
Please describe your approach to solving a complex problem
that requires both analytical thinking and creative solutions.
Consider multiple perspectives and potential obstacles.
How do you balance thoroughness with efficiency?
"""

# Run basic assessment
result = framework.assess_personality(sample_text)
print(f"Big Five Personality Scores: {result.personality_scores}")

Comprehensive Assessment Protocol

Setting Up Scientifically Validated Tests

For accurate results, implement the full assessment protocol with multiple parameter variations:

# Configure comprehensive testing parameters
config = {
    "param_combinations": {
        "temperature": [0.1, 0.2, 0.5, 0.7, 0.9],
        "top_p": [0.1, 0.5, 0.9],
        "presence_penalty": [0.0, 0.5, 1.0],
        "frequency_penalty": [0.0, 0.5, 1.0]
    },
    "stress_tests": [
        "adversarial_prompts",
        "contradiction_scenarios",
        "high_complexity_reasoning",
        "emotional_contexts"
    ],
    "cognitive_trap_tests": {
        "bias_identification": True,
        "reasoning_fallacies": True,
        "inconsistency_detection": True
    },
    "min_iterations": 1000,
    "confidence_threshold": 0.85
}

# Run comprehensive assessment
comprehensive_result = framework.assess_personality(
    text=sample_text,
    parameters=config
)

Note: The comprehensive assessment will run thousands of iterations to establish statistically significant personality profiles. This process may take several hours depending on the number of parameter combinations and test iterations.

Understanding Results

AI personality assessment results differ from human assessments due to the nature of LLMs:

  • Parameter Sensitivity: AI personalities can vary significantly with different temperature and sampling settings
  • Context Dependency: Personality expressions may change based on conversation context or role-playing instructions
  • Stability Metrics: Assess how consistent personality traits are across different testing conditions
  • Elasticity Measure: How much an AI's personality can change while maintaining internal consistency
